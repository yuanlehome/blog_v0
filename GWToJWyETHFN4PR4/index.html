2021年秋招经验经历总结

发表于 2021-11-27 分类于 秋招总结 阅读次数：
本文字数： 5.3k 阅读时长 ≈ 5 分钟

基本情况

211本985硕，算半个科班，实习和项目都勉强算有，但都极其拿不出手。C++技术栈，八股文和算法题准备还算充分，由于相关书籍看的多，对于编程理解还算深刻。

由于秋招结束的还算早，大大小小的厂都拿了。就经常出没于各大秋招交流群，水群水了一段时间。自己也闲着无事维护了美团和字节已意向群（分别有一千多人）。所以下面先介绍一些水群时学习到的行业”黑话“。

  • hr：指人力资源小哥哥小姐姐，每个公司每个部门都有负责招聘的hr。一般会通过电话或微信的形式和你沟通面试事项；

  • hc：head count部门人头数，意味着该公司或部门还有几个人员缺口。因为有集邮大佬，一般意向书会超发；

  • 活水：内部转岗；

  • jd：job description工作描述，就是公司发布的岗位描述页面；

  • offer：前期指录用意向书，后期就是正式签的三方；

  • 白菜/sp/ssp：sp指special offer，ssp指special special offer，代表薪水的档位以及公司的诚意。一般是根据你的面试表现、学校学历、offer数量等综合实力认定的。比例大概为6:3:1，薪资差距一般在3k左右；

  • xy：是大家说薪水的方式。比如说`2416`，指一个月24k的基本（base）工资，一年如果绩效达标发16个月薪水；

  • base：有两层含义，根据语境可以是城市，即岗位工作地点，也可以是上面说的基本工资。可能得根据语境判断意思：

    • 「你意向base哪儿」，是问你的工作地点
    • 「你base有多少k」，问薪水，如24*16

  • oc：指offer call，指给你发录用意向电话或邮件；

  • a：两层意思，第一层是argue的缩写，指要求更高的薪水，比如本来是白菜的档位，你不满意，可以要求提升到sp的档位，即要求加薪。一般在10月份（秋招）谈薪的时候。比如，大家说：

    • 「字节能a腾讯吗」，「会不会把offer a没了」等等

    第二层是accept的缩写，指代码题做出来了。大家会说：

    • 「这题你a出来了吗」，「我笔试全a」等等

  • 倒挂：指后入职的新人薪水比老员工高，互联网行业基本年年倒挂；

  • crud：指增删改查，即实现常见的业务需求接口，后台开发程序员经常吐槽自己天天在crud，做些没有技术含量的工作；

  • kpi：指关键绩效指标。有人说「某公司有刷kpi的行为」、「我被kpi了」，指某些公司有每年投递和面试的硬性指标要求，所以经常会拉很多人面试（就算知道你不合适，不想要你），面完就给你挂掉的行为；

  • 361：指绩效人数划分比例，实行末尾淘汰制度，把绩效划分为三档，30% 的人绩效最好，一定有10%的人绩效不合格；

  • 325：指阿里的绩效评分3.25，4分最高，3.25最低，3.25直接没有年终奖和薪水普调；

  • 交叉面：指别的部门的人来面试，一般在自己面试的部门完成两到三轮面试后可能会出现，在腾讯中有一个组织叫做面委会，校招中主要针对WXG（微信事业群）进行加试；

  • mentor：指工作中的导师；

  • leader：指部门主管，一般是面试环节的最后一轮的面试官。

秋招战况

整个面试过程，基本都是比较顺的，没有遇到那种捞了挂挂了捞的情况。以下公司根据收到意向书的时间顺序排列：

  • 字节，上海，商业化技术/后端转Go开发，部门负责抖音企业号和本地生活两大业务；
  • 浙江大华，杭州，服务器AI方向开发，好像需要用Python；
  • 科大讯飞，合肥研究院，AI平台相关C++开发；
  • 美团，北京/上海，平台/服务体验部/后台开发，做智能客服VoIP等音视频相关；
  • 百度，北京，AIG/深度学习技术平台部/深度学习框架架构研发，开发paddlepaddle的；
  • 网易雷火，杭州，游戏研发工程师，服务端方向；
  • 腾讯，深圳，CDG/金融科技/事务型开发，好像做香港微信支付相关后台开发。

我最后签了北京百度。

投递和面试经历

秋招共投递了20家公司，但实际参加了面试的也就一半，有的是我拒了面试，也有的是简历没过。笔试面试主要集中在8月份（7月末小试牛刀，9月初收尾）。先直观感受一下这密集的日程安排。

百度

百度的四大事业群（不全）：

百度秋招分为提前批和正式批，只有提前批可以内推或者定向内推，正式批没有内推的机制。因此，想去百度的话一定要投递其提前批。百度是属于挂了又捞的，一个部门挂了，过两天另一个部门还会捞你面试。

简历投递之后，在其官网可以看到自己的流程状态，简单介绍一下：

  1. 简历处理中：刚投递完就是这个状态；
  2. 部门筛选中：说明你的简历被分配到了某个部门，该部门hr正在筛选你的简历；
  3. 面试流程中：说明你的简历通过了筛选，接下来等待你的是三轮面试，每一轮都有可能挂掉；
  4. 简历共享中：面试没有通过，即挂了。挂的原因有很多，但接下来能做的只有等待，等其他部门捞你面试。这个时候，过来人的经验告诉你，如果公司有熟人（师兄师姐等等），可以联系他们的部门捞你。
  5. 面试通过：恭喜你，面试通过，收到意向书。

百度的面试一般分三轮，前两轮一定是技术面（基本是项目+八股+手撕代码+反问）。最后一轮是主管面，根据面试官的喜好，可以是技术面也可以是hr面，也即是说，可能会问技术问题，也可能会问成长经历、职业规划等hr问题。

我是7.6号投递的提前批，由于当时倾向于去上海，就投递了base上海的，估计是上海太卷了（指hc太少），没约我面试，直接简历挂。挂了之后就学乖了，把base改成了北京，然后就顺利多了。

面试经历：简历处理中 -> 部门筛选中 -> 简历共享中 -> 联系了一位师兄捞我（感谢感谢感谢） -> 部门筛选中 -> 面试流程中 -> 三面完成通过后 -> 岗位不满意，拒了换部门和岗位重面（这里联系了另一位师兄，感谢感谢感谢） -> 面试流程中 -> 面试通过。

原部门岗位：IDG测试部/测试开发工程师。连续三个工作日面完，hr口头告知通过，但一直没发意向书。

新部门岗位：AIG/深度学习技术平台部/深度学习框架架构研发工程师。连续两天走完三面，三面主管当时告知通过，第二天发来了意向书。

阿里

我是7.23号投递的，阿里一次可以投递三个平行志愿，我当时就随便投了两个，其中一个直接简历挂，另一个至今还在简历评估中。

阿里的笔试分很多批，投完简历之后任选一次参加即可。而且面试和笔试是分开的，就是说在笔试之前也可能发起面试。笔试时间一小时两道题，我当时是全a的。

至今在简历评估中的部门是菜鸟，估计不招C++的吧。建议大家投递之前搞清楚再投递。

后来九月底甚至十月份阿里有两个部门分别来了电话要约我面试，因为我当时已经有意向公司了，再加上令人不爽的投递和简历筛选经历，我就直接拒绝了。

腾讯

先简单介绍一下腾讯的六大事业群。

  • WXG

    负责微信生态体系的搭建和运营，依托微信基础产品，以及微信公众平台、小程序、微信支付、企业微信、微信搜索等开放平台，为各行各业的智慧化升级提供解决方案和连接能力。同时负责QQ邮箱、微信读书等产品的开发及运营。

  • IEG

    负责公司游戏、电竞等互动娱乐业务的研发、运营与发展。诵过系统性研究及前沿科技的探索和应用，不断为户创造更高品质的互动娱乐内容体验，助力公司在全球互动娱乐领域取得领先地位，并朕动公司相关业务共建繁荣向上的内容生态。

  • CDG

    CDG作为公司新业务孵化和新业态探索的平台，负责推动包括基础支付、金融应在内的金融科技业务、广告营销服务等重要领域的发展和创新。同时作为专业支持平台，为公司及各事业群提供战略规划、投资并购、国际拓展、市场公关等专业支持。

  • TEG

    负责为公司及各事业群提供技术及运营平台支持，负责研发管理和数据中心的建设与运营，并为用户提供全线产品和客户服务。同时牵头腾讯技术委员会，通过内部分布式开源协同，加强基础研发，建设技术中台等措施，支持业务创新。

  • CSIG

    负责推进公司云与产业互联网战略，依托云、AI等技术创新，打造智慧产业升级方案，探索用户与产业的创新互动，助力零售、医疗、教育、交通等产业数字化升级，同时协助企业更智能地服务用户，构建连接户与商业的智能产业新生态。

  • PCG

    负责公司互朕网平台和内容文化生态融合发展，整合QQ、QQ空间等社交平台，和应宝、浏览器等流量平台，以及新闻资讯、视频、体育、直播、动漫、影业等内容业务，推动IP跨平台、多形态发展，为更多户创造海量的优质数字内容体验。

以上都是官话，经过多方（脉脉）了解，受欢迎程度排名如下：WXG/IEG（卷且有钱） > TEG/CDG（不卷且没钱） > CSIG/PCG（卷且没钱）。

腾讯也是典型的挂了捞、捞了挂的机制，捞你的部门不确定。腾讯的投递，需要注意的是，一开始会让你选一个意向事业群，你的简历会优先被该事业群的部门看到，面试也是该部门先发起。如果挂了，之后，所有事业群都会看到你的简历并发起面试，所以之后去哪儿就由不得你了。

投腾讯的时候我有点怂了，就投了一个中规中矩的部门（指面试难度一般，HC一般，适合养老），CDG/金融科技岗，后台开发。

腾讯的笔试和面试也是分开的，投递之后简历筛选通过，就会发起面试，笔试可以不做（真的有点难度）。

我共经历了三轮面试，两轮技术面+hr面。如果投递的是WXG的话，还会多两轮面委面（就是交叉面），可见面试难度会更高。其中，腾讯的测评和云证（就验证一下你的身份证）一般会在hr面前后。就我而言，时间跨度为三周，每周面一轮，第三周hr面后两天就发来了意向书。一般情况下，云证完等十天半个月后oc或者挂掉。

美团

美团先笔试，笔试通过才约面。笔试是两小时五道题，难度中等吧。一般来说，a出四道绝对进面试（其它不了解）。

美团一般是2+1，即两轮技术面一轮hr面。有的部门是3+1，多一轮技术面。一开始以为三轮技术面的可能是sp，后来发现和sp没有关系，和部门有关。

美团也是挂了会捞，捞你的部门也是不确定的。我是一把过，历时三周，一周面一轮，第三周hr面完后，第二天发来了意向。

从我维护的美团意向群里可以看到，到家/到店事业群招人非常非常多，其次是优选。

字节跳动

在上海的话，国内大厂的研发中心属字节规模最大了。字节提前批免笔试，正式批需要笔试。也是典型的挂了捞，因此，早投递，机会更多。

字节的面试一般是三轮，三轮都是技术面，一般来说，一面面试官是同事，二面面试官是其它部门的（也叫交叉面），三面面试官是部门主管。每轮都要至少手撕两道编程题。总的来说，字节的面试更注重代码能力。

我在提前批启动几天后就投递了。一开始投递的是后端开发工程师-教育业务。投完第三天就约面了，第一面是校友，聊的就很愉快，这也给我后面的面试定下了基调。后面就都很顺利，就是时间跨度比较大，二面和一面隔了整整半个月（一面完当天就约了二面，由于面试官时间的原因）。三面完，如果通过的话，有个等待offer审批的过程，时间不定，一般都要一周多才出结果，我就等了10天才收到意向书。

一周后，我放弃了这份意向书，主要是因为教育业务的原因，双减政策下来后，业务前景堪忧。后来，通过水群，从一位老哥那里知道了一个小道消息：拿了意向书之后，不满意的话，可以联系其它部门的hr申请直通终面。好家伙，之前一直犹豫不决就是因为怕重走一次三轮面试再挂了，这竟然有机会直接终面？！所以，在拿到美团意向书之后，当天，我就拒了教育业务的，申请了商业化技术的直通终面，终面就是主管面，同样也是顺利通过了。

网易（雷火）

雷火的笔试，或者说网易的笔试，出了名的又臭又长，三个小时五道编程题，难度基本中等偏上。我觉得是所有互联网公司笔试中最难的了。

雷火的面试共四轮，其中一面有两轮平行面+一轮主管面+hr面。雷火的面试偏语言细节，基本是给很多段代码，让你分析运行结果，分析错误等等。因此，需要对C++语言本身掌握扎实。

不过，雷火面试体验还挺好的。

拼多多

投了提前批和正式批，都是笔试完就挂（笔试成绩拿了80%以上的分，还是挂）。笔试又臭又长又难，耽误时间。不说了，都是泪。

大华

大华笔试不是通过牛客，就是自己的网站，主要是选择题和两道编程题（不用运行，编出来就行）。面试是两轮技术面+一轮hr面，都是电话面，每一轮大概半小时。二面面试官有点东西的，问的问题有点深度。

总的来说面试体验还不错，推荐投递。

米哈游

今年米哈游异常的卷。感觉有上万人投递，hc就600+？估计因为原神爆火的原因吧，反正校招群里一片哀嚎。。。

米哈游提前批免笔试，正式批需要笔试。

深信服

一面挺顺利。二面挂了，当天晚上收到感谢信。效率很高！

后面的公司有的投了没反应，有的约面比较晚就拒了面试，就没怎么可记录的了。包括海康威视、同花顺、vivo、TP-LINK、京东、中兴、BIGO、虎牙直播、oppo等。

秋招心得

零零散散的说一下。

  • 一定要尽早投递，投的越早越容易；
  • 前期建议拿小厂练手；
  • C++选手建议首选百度腾讯，Java首选阿里美团，但去了字节基本都要转Go；
  • 宁去中厂核心不去大厂边缘，不要太有大厂情结；
  • 去某公司某部门之前在脉脉上问问有没有坑，虽然脉脉上除了Google微软没有能去的；
  • 貌似大厂实习转正拿sp的概率大很多；
